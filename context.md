# Context
This document tracks the current state of the amna_OS project.

- The application is a cyberpunk/vaporwave themed interactive desktop OS.
- It includes a boot screen, taskbar, interactive windows, terminal, and various easter eggs.
- Typing "mohib" in the terminal transitions to a "second screen" (easter egg) with a 3D alien cat, synchronized lyrics, and floating GIFs.
- Current fixes applied:
  - Lyrics synchronized strictly to the timestamp intervals in `lyrics.txt` so they line up precisely with the song.
  - Grey boxes and sparkles removed.
  - 3D cat scaled up 1.8x.
  - Thought bubble realigned to rest properly on top of the cat model.
  - GIFs restored to float around when the `mohib` easter egg triggers.
